# EstimaPro-Sprint1

# El presente documento describe la planeación del desarrollo del proyecto EstimaPro, una aplicación web orientada a centralizar la toma de requerimientos y automatizar estimaciones técnicas para desarrolladores freelance y pequeñas agencias.
El desarrollo se ejecutará bajo la metodología ágil Scrum, estructurando el trabajo en sprints de 2 semanas con entregables funcionales al final de cada iteración.
